Nesse projeto visa monitorar as condições do ambiente atravez de um sensor DHT11 usando o código fonte ESP32.
Nesse projeto é coletado a Temperatura, Umidade e Luminosidade (LDR) pelo ESP32 e transmite os dados diretamente para o mqtt ou MyMqtt para o celular, onde tais dados são coletados via wi-fi
Componentes usados foram: ESP32 (Microcontrolador responsável pela leitura dos sensores), Sensor DHT11 (Mede a temperatura e a umidade do ambiente), LDR (Detecta a intensidade da luz ambiente), Wi-Fi e um Dashboard HTML
